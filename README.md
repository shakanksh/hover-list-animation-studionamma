# Hover List Animation

A recreation of the hover interaction from [Studio Namma](https://studionamma.com/) — hovering a row reveals its paired image via a clip-path animation while the other headings fade out. Built twice using two different animation approaches.

## Implementations

- `/gsap` — imperative, using GSAP + `useGSAP`
- `/motion` — declarative, using Motion for React

## Stack

- Next.js
- Tailwind CSS
- GSAP + @gsap/react
- Motion

## Running locally

```bash
npm install
npm run dev
```
